# High-Level Architecture

## Objective

Build a production-grade AWS 3-tier infrastructure that is highly available, highly scalable, secure, and automated.

## Core Components

- Amazon Route 53
- Application Load Balancer
- EC2 Auto Scaling Group
- Amazon RDS Multi-AZ
- Bastion Host
- NAT Gateway
- Internet Gateway
- CloudWatch
- SNS
- SQS
- IAM
- Terraform
- Docker
- Ansible

- Load Balancing
