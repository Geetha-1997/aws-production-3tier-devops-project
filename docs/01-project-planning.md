# Project Name

**Production-Grade Highly Available & Highly Scalable AWS 3-Tier DevOps Project**

---

# Project Objective

Design and deploy a production-ready AWS infrastructure that is secure, scalable, highly available and follows AWS Well-Architected Framework principles.

---

# Business Scenario

ABC Technologies is a rapidly growing e-commerce company.

The company currently hosts its web application on a single server.

During festivals and seasonal sales, website traffic increases dramatically.

The existing infrastructure experiences:

- Downtime
- Performance degradation
- Manual deployments
- Security risks
- No disaster recovery
- No monitoring
- Poor scalability

To overcome these challenges, the company plans to migrate to AWS and implement a production-grade 3-tier architecture using DevOps best practices.
